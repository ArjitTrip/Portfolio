Nexo:- Local System Assistant

Nexo AI is a desktop assistant designed to provide a secure, modern, and intuitive way to interact with your local computer. At its core, the assistant is protected by a biometric security layer, utilizing facial recognition to grant access. Once authenticated, users can execute commands to seamlessly open files, launch applications, and perform other system tasks. The project is built with Python, leveraging powerful libraries for computer vision and system control..

Real-time Currency calculator

A real-time currency calculator built using live exchange rate APIs to convert values between multiple global currencies instantly. The application fetches up-to-date rates from an external API and performs accurate conversions with a clean, responsive interface. Users can select base and target currencies, input amounts, and view results dynamically without page reloads. Tech Stack: HTML, CSS, JavaScript, REST API (Live Exchange Rate API)
